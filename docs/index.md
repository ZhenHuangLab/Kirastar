# KIRASTAR - 深空摄影系列笔记

这是一本关于天文摄影的笔记，主要是系统性地记录我在学习天文摄影过程中的一些心得以及体会，方便对这方面感兴趣的天文爱好者进行参考及查阅；同时，也是我送给浙大天协的一份礼物，感谢四年的本科时光有你们的陪伴。如果没有浙大天协的伙伴们，就没有现在的我。
